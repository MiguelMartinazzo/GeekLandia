# GeekLândia

O **GeekLândia** é um website desenvolvido com foco na cultura geek, criado como projeto acadêmico para a disciplina de **Desenvolvimento de Framework II**.

O projeto tem como objetivo aplicar os conhecimentos adquiridos em **HTML**, **CSS** e **JavaScript**, desenvolvendo uma aplicação web visualmente atrativa, funcional e responsiva.

## Sobre o Projeto

O GeekLândia é um site voltado para a publicação de conteúdos, artigos e informações relacionadas ao universo geek. A proposta do projeto é oferecer uma experiência simples e intuitiva para os usuários, reunindo páginas informativas, listagem de produtos/serviços e uma simulação de compra.

Durante o desenvolvimento, foram trabalhados conceitos fundamentais de estruturação web, estilização responsiva e funcionalidades dinâmicas com JavaScript.

## Funcionalidades

O projeto conta com as seguintes funcionalidades:

- Página inicial com apresentação do site;
- Publicação de conteúdos relacionados à cultura geek;
- Página de cadastro de cliente;
- Validação de login;
- Listagem de produtos e serviços;
- Página de detalhes de produto;
- Simulação de finalização de compra;
- Página de compra finalizada;
- Layout responsivo para diferentes dispositivos;
- Botão para retornar ao topo da página;
- Barra superior fixa durante a rolagem da página;
- Manipulação dinâmica de elementos com JavaScript.

## Tecnologias Utilizadas

As principais tecnologias utilizadas no desenvolvimento foram:

- HTML5
- CSS3
- JavaScript
- PHP
- Netlify
